Remove Powered By Odoo

APERÇU

Ce module vous permet de supprimer "Powered By Odoo:

  • De tous les modèles de mail et supprimer
  • Du site web

  • Fonctionnalités
  • Capture d'écran
  • Besoin d'aide
Supprimer généré par Odoo de tous les modèles de mail.

Installer le module

Dans Configuration --> Technique (Activer le mode développeur pour voir ce menu) --> Modèles d'email --> Sélectionner tous les modèles --> Action --> Supprimer Powered By Odoo

Remove powered by odoo from email templates

Install the module

On the Settings --> Technical (activate the developer mode to see this menu) --> Email Templates --> Select all the templates --> Action --> Remove Powered By Odoo

Note: If you are using odoo in many languages repeat the previous instruction for each language .
